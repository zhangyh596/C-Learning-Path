## 📖 简介 (Introduction)
你好！我是 **小张**。
这个仓库用于记录我在学习 C 语言的全过程。
这里存放了我从零基础开始的所有代码，包括基础语法练习、数据结构手写实现（顺序表、链表、哈希表等）以及基于这些数据结构的实战小项目。

## 📂 仓库目录规划 (Directory Structure)

我将按照学习进度，依次完善以下模块：

- **`00_Basic_Exercises/`**
  - 🗃️ **归档区**：存放早期的语法练习代码（循环、函数、数组、指针初阶等）。
  - 记录了我从 Hello World 开始的点点滴滴。

- **`01_SeqList/`**
  - 📚 **顺序表 (Sequential List)**：
  - 手写实现动态增长的数组。
  - 核心功能：动态扩容、头插、尾插、头删、尾删、查找。

- **`02_Contact_Seq/`**
  - 📒 **通讯录 (顺序表版)**：
  - **底层依赖**：基于 `01_SeqList` 顺序表实现。
  - 功能：实现联系人的增删查改、排序、打印等功能。

- **`03_SList/`**
  - 🔗 **单向链表 (Singly Linked List)**：
  - 相比顺序表，链表在任意位置插入删除更高效。
  - 重点攻克：结构体指针、二级指针传参、节点内存管理。

- **`04_Contact_SList/`**
  - 📒 **通讯录 (单链表版)**：
  - **底层依赖**：基于 `03_SList` 单链表实现。
  - 改进：解决了顺序表需要连续内存空间的限制，实现按需申请内存，更加灵活高效。

- **`05_DList/`**
  - 🔗 **双向带头循环链表 (Doubly Linked List)**：
  - 链表结构的完全体！
  - 核心优势：解决了单链表找前驱节点极其麻烦的痛点，增删查改效率极高。

- **`06_SnakeGame/`**
  - 🐍 **贪吃蛇小游戏 (Snake Game)**：
  - **实战项目**：综合运用 C 语言语法与数据结构（链表）开发的经典小游戏。
  - 重点攻克：控制台控制、游戏主循环、坐标操作。

- **`07_HashTable/`**
  - 🗄️ **哈希表 (Hash Table)**：
  - 手写 C 语言版工业级哈希表。
  - 核心技术：djb2 哈希算法、拉链法解决冲突、**动态扩容 (Rehashing)** 维持 O(1) 查找速度、零内存泄漏处理。
